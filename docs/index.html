View and modify

Most common image formats can be loaded (including DICOM) and viewed in the application

Modify k-space with a multitude of parameters commonly used scanner software. See the effects on the reconstructed image immediately

Simulate simple acquisitions to see how an image forms and evaluate the difference of k-space filling methods

Download

Free and open source

The software and its dependencies are open source and freely available - anyone can view and contribute on GitHub

For many students the concept of k-space is a difficult topic of MR physics. Understanding can be facilitated by simple visualisation

Short usage instructions are available on the GitHub page where bugs and feature suggestions can also be discussed
